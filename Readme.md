# RESTful API Sederhana dengan Go + Fiber

> **Belajar membangun REST API cepat dan efisien menggunakan Golang dan framework Fiber**  
> _Dibuat pada: 16 November 2025_

---

## Ringkasan Proyek

| Informasi | Detail |
|---------|--------|
| **Nama Proyek** | `belajar-fiber-api` |
| **Bahasa** | Go (Golang) |
| **Framework** | [Fiber](https://gofiber.io) v2+ |
| **Database** | PostgreSQL (via `database/sql` + `pgx`) |
| **ORM / Query Builder** | `goqu` (opsional) |
| **Fitur Utama** | CRUD Customer, Validasi, Error Handling |
| **Tujuan** | Belajar arsitektur clean, dependency injection, dan performa tinggi |

---

## Tech Stack

```yaml
- Go: 1.21+
- Fiber: Web framework berkecepatan tinggi
- PostgreSQL: Database relasional
- goqu: SQL builder (opsional)
- Docker: Containerization
- VS Code: Editor utama
